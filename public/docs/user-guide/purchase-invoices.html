Purchase Invoices

Purchase invoices are the invoices you receive from suppliers — electricity, phone, office supplies, professional services, and so on. This is where you upload them, check what Merlin extracted, and post them to your accounts.

Uploading invoices

Merlin accepts PDF files, Word documents (.docx), Excel spreadsheets (.xlsx), and CSV files.

  1. Go to Purchase Invoices.
  2. Drag one or more files onto the upload area, or click it to open a file picker.
  3. Merlin reads the files and processes them in the background. Each invoice appears in the list once processing is complete.

You can also set up a watch folder — a folder on the server that Merlin monitors — and drop files into it to have them picked up automatically. Ask whoever manages your Merlin installation to set this up.

What Merlin extracts

For each invoice, Merlin tries to find:

  • The supplier name and tax number
  • The invoice number and date
  • Each line item: description, quantity, unit price, and VAT amount
  • The invoice total

It also suggests which account each line should be posted to, based on how similar invoices have been posted in the past.

Reviewing an invoice

Click an invoice in the list to open the review panel.

Check the following:

  • Supplier — is Merlin's match correct?
  • Invoice number and date — do they match the document?
  • Line items — are the descriptions, amounts, and VAT correct?
  • Account allocation — is each line posted to the right account? Click the account on any line to change it.

You can edit any line item directly in the panel. Click a field to change it and save your changes.

Actions

Once you've reviewed an invoice, use the action buttons to move it forward:

  • Mark as Reviewed — records that the invoice has been checked. It stays in the queue for approval.
  • Approve — approves the invoice for posting.
  • Post — posts the invoice to your accounts. This is the final step.
  • Dispute — flags the invoice as disputed if there's a problem with it (e.g. wrong amount, duplicate). You can still post it later once resolved.
  • Reject — rejects the invoice. Use this if the invoice should not be posted at all.
Which actions you can take Your access level determines which actions are available to you. If a button is missing, you may not have the required permission. Ask your administrator to check your role settings.

Automatic posting

If you have posting rules set up for a supplier, and Merlin is confident enough in the extraction, it will post the invoice automatically without any review. You'll see these invoices arrive in the list with a Posted status.

The confidence threshold for automatic posting is set in Settings.

If the extraction went wrong

If Merlin misread an invoice — wrong supplier, garbled line items, missing amounts — use the Reprocess button. This deletes the extracted details and sends the invoice through the reading process again from scratch. Reprocessing is useful if the file was initially unreadable and has since been replaced, or if a settings change should change how it's extracted.

Viewing the original file

You can view the original uploaded file from the review panel at any time to compare against what Merlin extracted.
