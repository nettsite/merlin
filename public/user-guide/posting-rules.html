Automatic Posting

Posting rules let you tell Merlin to process certain invoices automatically — no review needed. Once a rule is set up, invoices that match it are read, extracted, and posted to your accounts without anyone needing to touch them.

This is most useful for regular, predictable invoices: monthly subscriptions, recurring utility bills, or any supplier whose invoices always follow the same pattern.

How rules work

When Merlin finishes reading an invoice, it checks whether any active posting rules match. A rule matches when:

  • The supplier matches (if the rule is set to a specific supplier), and
  • All the conditions you set are met (total amount, confidence level, line item keywords), and
  • Merlin's confidence in the extraction is above the threshold in Purchasing Settings.

When a rule matches, Merlin carries out the actions you configured — which can include automatically approving the invoice and posting it.

Setting up a rule

  1. Go to Posting Rules in the navigation.
  2. Click Add Rule.
  3. Give the rule a clear name, such as "Acme Plumbing — Monthly Retainer".
  4. Optionally, choose a specific supplier. If you leave this blank, the rule applies to any supplier.
  5. Set conditions to control when the rule fires (see below).
  6. Set actions — tick Auto-approve and/or Auto-post.
  7. Set the rule to Active and save.

Conditions

Conditions limit which invoices a rule applies to:

ConditionWhat it does
Minimum confidenceThe rule only fires if Merlin's confidence score for the extraction is at or above this number. Set higher to be more cautious; lower to be more permissive.
Total range (minimum)The invoice total must be at least this amount.
Total range (maximum)The invoice total must be no more than this amount.
Line description containsAt least one line item description must contain this text. Useful for matching invoices by what they're for.

You can leave any condition blank — it simply won't be checked. An invoice needs to satisfy every condition you set for the rule to fire.

Actions

ActionWhat it does
Auto-approveMerlin approves the invoice automatically without anyone clicking Approve.
Auto-postMerlin posts the invoice to your accounts automatically. Most rules will have this turned on.

Checking that rules are working

The rule list shows a Match count and Last matched date for each rule. If a rule has a zero match count for recent invoices that should have triggered it, check:

  • Is the rule set to Active?
  • Does the supplier on recent invoices match the supplier set on the rule?
  • Is the total within any range conditions you set?
  • Is Merlin's confidence on recent invoices above the minimum confidence on the rule? (You can check this in Purchase Invoices by looking at the confidence shown on each invoice.)

Pattern matching for repeat invoices

Beyond rule conditions, Merlin also checks whether a new invoice from a supplier closely matches their most recently posted invoice — similar amounts and similar line descriptions. If it matches well enough, Merlin treats that as additional confidence to post automatically, even without a posting rule.

The tolerance for "close enough" is set in Purchasing Settings.
